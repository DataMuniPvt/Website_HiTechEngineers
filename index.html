Current Leak Detection and Alert System

Electric Pole Monitoring System

The system is one of it's kind solution to proactively curb current leakage problems and prevent any accidents like electrocution from happening

This device can sense as low as 30mA of current and send the signal to the user on leakage so that appopriate measures can be taken. This device works in real-time and the supports multiple modes of communication.

It conists of a ring type CT and wires through it which are used for leakage detection. It's main purpose is to detect flow of current where current flow is unwanted, as in earth wire or neutral wiring systems.

Some of its application -

  • Current leakage detection at Streetlight poles.
  • Protection system at construction sites.
